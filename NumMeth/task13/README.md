Дан правильный шестиугольник. Он разбивается на треугольники. После каждого шага каждый треугольник делится на четыре по серединам сторон. На полученной сетке построить аппроксимацию функции

f_h(x, y) = sum_i c_i * phi_i(x, y),

где phi_i — глобальные кусочно-линейные базисные функции.

Коэффициенты определить:

1) интерполяцией: c_i = f(P_i);

2) методом наименьших квадратов.

Построить графики исходной функции, аппроксимации и ошибки. Исследовать зависимость ошибки от уровня разбиения n.
